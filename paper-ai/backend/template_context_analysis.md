## 后续规则建议

1. 模板说明段落过滤：包含“如图”“如表”“如文献”“编号”“示例”“格式”等组合时，引用类 high_risk 应降级。

3. 双语题名合并：相邻 `表1` / `Table 1` 或 `图1` / `Figure 1` 应视为同一图表的中英文标题。
4. 附录样例降级：`图A1`、`表B2`、`文献[A3]` 在编号规则说明段落中应按附录样例处理。
