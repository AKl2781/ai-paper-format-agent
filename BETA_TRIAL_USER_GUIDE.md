# Beta Trial User Guide

这是一个测试版工具，目前主要用于论文或报告类 `.docx` 文档的格式检查和格式修改。

它不是论文代写工具，也不会替你生成论文正文。

## 使用前请先了解

- 当前是测试版，不保证所有论文都能完美处理。
- 目前只建议上传 `.docx` 文件。
- 暂不建议上传隐私敏感文件、涉密文件、正式最终提交版论文。
- 建议先复制一份备份文件，用备份副本测试。
- 输出结果需要你自己检查，确认无误后再使用。

## 当前支持什么

当前主要支持：

- 文档类型识别。
- 格式检查。
- 格式修改。
- 修改报告。
- 在线预览。
- 下载修改后的 `.docx` 文件。
- 重复风险检测和相似度预检。

## 当前不承诺什么

当前不承诺：

- 不承诺论文代写。
- 不替用户生成论文正文。
- 不保证深度论文润色。
- 不保证所有学校模板都完全匹配。
- 不保证在线预览和 Word/WPS 打开效果完全一致。
- 不提供知网、维普、万方查重服务。

## 建议测试流程

1. 准备一份备份 `.docx` 文件。
2. 上传文件。
3. 查看系统识别的文档类型。
4. 运行格式修改。
5. 查看修改报告。
6. 查看在线预览。
7. 下载修改后的 `.docx`。
8. 用 Word 或 WPS 打开下载文件。
9. 检查标题、正文、图片、表格、参考文献等是否正常。

## 不符合预期时请反馈

如果输出不符合预期，请尽量反馈：

- 原文件类型，例如论文、课程报告、实验报告。
- 文档大小。
- 页数。
- 是否使用模板。
- 问题截图。
- 错误提示。
- 你看到的具体问题。
- 是否能成功下载文件。
- 下载后的文件能否正常打开。

## 建议不要上传的文件

暂不建议上传：

- 含有个人隐私的文件。
- 含有未公开研究数据的文件。
- 含有单位或学校敏感信息的文件。
- 正式最终提交版论文。
- 非 `.docx` 文件。
- 你没有权利测试或分享的文件。

## 试用目标

这次试用主要想确认：

- 是否能正常上传。
- 是否能正常生成修改版。
- 是否能正常预览。
- 是否能正常下载。
- 下载后的 Word 文件是否能打开。
- 格式修改是否有帮助。
- 是否出现图片、表格、正文丢失。
- 修改报告是否容易理解。

谢谢参与测试。你的反馈会直接决定下一版优先修什么。
